# A2A-SIN-Teams

> A2A agent for Teams integration within the OpenSIN ecosystem.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Stars](https://img.shields.io/github/stars/OpenSIN-AI/A2A-SIN-Teams)](https://github.com/OpenSIN-AI/A2A-SIN-Teams)

## Overview

This repository provides an Agent-to-Agent (A2A) integration for Teams, enabling seamless communication and automation within the OpenSIN-AI ecosystem.

## Quick Start

```bash
git clone https://github.com/OpenSIN-AI/A2A-SIN-Teams.git
cd A2A-SIN-Teams
npm install
npm start
```

## Features

- A2A protocol support for Teams
- Automated message routing
- Integration with OpenSIN MCP servers
- Full observability via OpenSIN-Ledger

## Architecture

This agent connects to the OpenSIN orchestrator and handles Teams-specific messaging through the A2A protocol.
